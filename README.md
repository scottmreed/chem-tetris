# ChemAsciiTetris Discord Bot

A Discord bot that brings the addictive chemical-themed ASCII Tetris game to your server! Build molecules by arranging falling Carbon (C) and Oxygen (O) atoms to match target patterns. Challenge friends, climb leaderboards, and discover chemistry through gaming.

## Features

- **Chemical Tetris Gameplay**: Arrange falling atoms to build molecular structures
- **ASCII Art Interface**: Retro-style gameplay with clean, readable ASCII graphics
- **Multiplayer Support**: Challenge friends and compete on server leaderboards
- **Educational Fun**: Learn about basic molecular structures while playing
- **Persistent Scores**: Track high scores and game statistics

## Setup

1. Clone this repository
2. Install dependencies: `npm install`
3. Configure your Discord bot credentials in `.env`
4. Run the bot: `npm start`

## Environment Variables

Create a `.env` file with the following variables:

```
DISCORD_APPLICATION_ID=1471703357973336145
DISCORD_PUBLIC_KEY=a3d825725657c3d1b04f7c8f63d08cdf35fe2b87b6644a7164821650c49ae0c8
DISCORD_TOKEN=your_bot_token
```

## Discord Application Setup

You don't need to provide the Interactions Endpoint URL or Linked Roles Verification URL for basic bot functionality. These are optional advanced features for HTTP-based interactions and role verification.
